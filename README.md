# Personal Profile Page

This is a simple personal profile webpage built using HTML and CSS. It demonstrates fundamental web development skills including semantic HTML structure, various CSS display properties (flexbox, grid), text and font styling, basic responsiveness, and form implementation.

## Project Goal

The primary goal of this project was to practice and apply core web development concepts learned, focusing on:
•   Semantic HTML structure
•   CSS basics (display, Flexbox, Grid, text styling, fonts, backgrounds)
•   Responsive design principles

## Features

•   **Header Section:** Displays name and title.
•   **Profile Section:** Features a circular profile image and a short bio, laid out using Flexbox.
•   **Skills/Interests Section:** Showcases at least four items arranged using CSS Grid.
•   **Contact Form:** Includes fields for name, email, and message, with a submit button.
•   **Footer:** Displays copyright information.
•   **Responsive Design:** Adapts layout and readability for various screen sizes (mobile, tablet, desktop) using media queries.

## Technologies Used

•   **HTML:** For semantic structure and content.
•   **CSS:** For styling, layout (Flexbox, Grid), and responsiveness.
